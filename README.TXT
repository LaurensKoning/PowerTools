The PowerTools are a collection of tools for automated software testing.
They include an engine, engine extensions (e.g. for FitNesse) and interfacing
modules (e.g. for web application testing).

The copyright to all PowerTools material, including source code and
documentation, lies with Martin Gijsen (www.deAnalist.nl).
